# About
此项目是Forked from http://bookcase.chromeexperiments.com 
,是本人辛苦抓取该网页的源码，一般的整站抓取软件基本抓不全，要么文件目录很乱，
所以花了几个小时整理、修改一些代码，然后个人这一块不怎么熟练，所以比较费劲，
大家如果有好的方法或者流程可以分享给我。
效果是一个 无尽旋转的书架，点击一本书可以查看该书籍的介绍，
以及扫一扫可以跳转到谷歌书店的二维码，整个效果还是非常酷的，
起码在我网上找的几种书架效果中是最酷的。
